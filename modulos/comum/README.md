# Modulo Comum

Este diretorio concentra codigo reutilizavel pelos modulos `producao` e `precorte`.

Componentes candidatos para migracao:

- leitura de PDF;
- OCR de consumo;
- leitura de `timing`;
- parse e formatacao numerica;
- integracao com Excel via COM;
- utilitarios de caminhos e organizacao de arquivos.
